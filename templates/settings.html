Greeting Tool Prompt Templates

Title/Description Generation

System prompt for auto-generating titles and descriptions for greetings. The following macros are available to use:
  • {{existingTitles}} contains existing greeting titles to avoid duplicates
  • {{titleLength}} contains the configured title length instruction
  • {{descriptionLength}} contains the configured description length instruction

Greeting Generation

System prompt for generating new greeting messages. The following additional macors are available to use:
  • {{customPrompt}} contains user-requested theme/scenario if provided
  • {{existingTitles}} contains existing greeting titles to avoid duplicates
  • {{greetingLength}} contains the configured greeting length instruction
